## Challenge

The before kernel's inner loop performs a shared-memory load (smem_zij[accum_n * num_queries_in_block + accum_m]) with a 2D index computation and a global bounds check (global_query_idx < num_queries_in_block && global_key_idx < num_keys_in_block) for every element on every iteration


## Solution

The after kernel reads the shared-memory mask only once before the loop, compresses it into a single uint32_t bitmask register (dropout_keep_mask), and replaces the per-iteration smem load + branch with a single BITTEST instruction (dropout_keep_mask & (1u << i)).
