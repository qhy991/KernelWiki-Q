## Solution

Cast float pointers to P32* (array_t<float,4> with __align__(16)) so the compiler emits ld.128/st.128 instructions, loading 4 floats per transaction and reducing loop trip count and memory instruction count by 4x

## Key Lessons

- Use __align__(16) struct wrappers (array_t<T,16/sizeof(T)>) to cast scalar pointers and force the compiler to emit 128-bit ld.128/st.128 instructions — this consolidates 4 scalar fp32 transactions into one, cutting memory instruction count by 4x on bandwidth-bound elementwise kernels
- On memory-bandwidth-bound workloads (vector add, elementwise ops), the primary bottleneck is memory instruction count rather than compute; reducing transactions via wider loads/stores yields near-proportional speedups
