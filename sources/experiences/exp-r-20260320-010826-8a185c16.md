## Key Lessons

- Vectorization with float4 significantly improves memory bandwidth utilization
- Processing multiple elements per thread reduces kernel launch overhead
- Shared memory broadcast of scalar values avoids redundant global memory reads
- Fallback kernel needed for handling non-divisible dimensions
