## Solution

Replace scalar strided loads with AlignedArray<float, 4> vectorized packed loads (BYTES_PER_LDG=16), assigning VPT=16 elements per thread so each thread owns an entire row and loads it in a single LDG_PER_THREAD=1 16-byte transaction, maximizing memory coalescing and eliminating redundant load passes

## Key Lessons

- Use AlignedArray<T, N> with alignas to issue N-element vectorized loads in a single instruction — applies whenever elements per row is a power-of-2 multiple of the vector width, converting scattered scalar STG/LDG into packed 16/32-byte transactions
- Assign VPT (values-per-thread) >= num_cols so each thread owns an entire row, enabling thread-local reduction without cross-thread coordination and reducing total load instructions from 3 strided passes to 1 packed load
