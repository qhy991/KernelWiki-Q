## Key Lessons

- Diagonal matrix multiplication diag(A) @ B is equivalent to row-wise scaling
- For C[i,j] = A[i] * B[i,j], use 2D grid with thread blocks (e.g., 16x16)
- Each thread reads A[i] once and B[i,j] once, writes to C[i,j]
- Boundary checking: i < N and j < M
