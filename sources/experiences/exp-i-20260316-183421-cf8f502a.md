## Solution

## Key Lessons

- Shared memory layout must match actual head_dim (128), not BLOCK_M
- Dynamic shared memory allocation helps fit within 48KB limit
- Online softmax requires careful handling of m_prev, l_prev updates
- Mixed precision (FP32 query, F16 KV) needs explicit conversion
