## Key Lessons

- The after version eliminates all smemLogits writes, reducing per-block shared memory from ~20KB (smemLogits 8KB + smemIndices 8KB + histogram/cumSum 4KB + misc) to ~12KB (smemIndices 8KB + histogram/cumSum 4KB + misc).
- The final logits output in the after version reads directly from global input using smemIndices as a lookup table, which leverages L2 cache residency from the earlier histogram-building pass rather than requiring a write-then-read smem intermediary pattern.
