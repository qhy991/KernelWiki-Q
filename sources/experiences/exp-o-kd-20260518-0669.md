## Solution

sced memory transactions are required, under-utilizing the GPU's 128-byte memory transaction granularity and bandwidth.
Applied change: The gelu_f32x4_kernel vectorizes the load/store using float4 (128 bits), so each thread reads and writes 4 adjacent floats in a single coalesced memory transaction.
