## Solution

Extract the inline two-phase block reduction (warpReduceMax → shared memory inter-warp → warpReduceMax) into a __device__ __forceinline__ blockReduceMax() utility function, called from the kernel in a single line

## Key Lessons

- Extracting inline reduction patterns into __device__ __forceinline__ utility functions improves code maintainability and reusability without performance cost — the compiler inlines the call and generates identical PTX/SASS instructions
- When refactoring GPU kernels, use __forceinline__ on extracted device functions to guarantee zero abstraction overhead — applicable whenever duplicating reduction, scan, or scan-like patterns across multiple kernels
