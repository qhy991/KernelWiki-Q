KernelBench level_1 task_18: C = A.T * B.T matrix multiplication optimization

## Key Lessons

- Shared memory tiling provides modest speedup (1.03x) for transposed matmul
- TILE_SIZE=32 may not always be better - depends on problem dimensions and memory access patterns
- For M=1024, N=2048, K=4096, TILE_SIZE=16 is more stable than TILE_SIZE=32
- Correctness is the primary requirement - speedup is optional
