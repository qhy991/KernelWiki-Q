KernelBench level_1 task_9: Tall-and-skinny GEMM C = A @ B where A(16384,16) @ B(16,16384) = C(16384,16384). Inner dimension K=16 is very small.

## Key Lessons

- For GEMM with very small inner dimension (K<=16), simple thread-per-element is efficient
- Shared memory tiling overhead not worth it when K is small
- 16x16 thread blocks provide good occupancy for this workload
- Correctness achieved with max_diff=0.0
