## Challenge

The before code reads input using scalar strided loads in a loop: `float val = input[thread_row_offset + ii]` where ii increments by BLOCK_SIZE (256). For num_experts=16, only 16 threads per block carry real data; the remaining 240 threads have no work. Each load instruction fetches only 4 bytes ...


## Solution

The after code uses AlignedArray<float, 4> (BYTES_PER_LDG=16) to load 4 floats in a single 16-byte aligned load instruction.
