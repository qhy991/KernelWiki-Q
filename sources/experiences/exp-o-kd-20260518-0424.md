## Key Lessons

- Since the innermost dimension has stride=1, all 32 lanes then perform contiguous 128-bit vector loads from that base, achieving fully coalesced access within each head's 512-element nope data.
- The before kernel's grid-stride loop has especially poor memory access patterns here: adjacent threads in a warp access different (token, head) pairs with widely separated stride offsets, producing non-coalesced scalar loads across the 32768-byte head stride.
