## Challenge

Skeleton kernel needs to create a local cutlass::Array<T, N> variable, zero all its elements, and store the result to global memory at a per-thread offset using the correct cutlass Array API


## Solution

Declare a local cutlass::Array<T, N> storage, call its built-in clear() member to zero all N elements, then assign directly to ptr[threadIdx.x] which leverages cutlass::Array's operator= for the global memory write

## Key Lessons

- cutlass::Array<T, N> provides a built-in clear() method that zero-initializes all N elements regardless of type T — prefer it over manual memset or loop-based zeroing for correctness and clarity
- Direct assignment ptr[threadIdx.x] = storage works for cutlass::Array global memory stores because the type's operator= handles the element-wise copy, no explicit memcpy or store intrinsic needed
