## Challenge

Optimization focus: Replace single hardcoded int8 SM90 config with five M/N-aware config structs and a dispatch function.
Problem before: The SM90 int8 quantized GEMM path in cutlass_scaled_mm_sm90 used a single hardcoded configuration for all problem sizes: TileShape=_128x128x128, ClusterShape=_1x2
