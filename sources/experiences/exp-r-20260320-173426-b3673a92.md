## Key Lessons

- Vectorized loads (float4) require 16-byte alignment which PyTorch tensors don't guarantee
- uint4 also has alignment requirements on modern GPUs
- For PyTorch extensions, scalar loads with coalesced access patterns are safer and more portable
- Shared memory tiling + loop unrolling provides good performance without alignment complexity
