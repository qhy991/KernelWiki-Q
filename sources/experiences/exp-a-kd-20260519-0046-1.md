## Solution

Replace if-else branch with PTX predicated move instructions using inline assembly (.reg .pred p; setp.ne.b32 p, clear, 0; @p mov.u32 m, 0) to eliminate explicit branch instructions and use predicate-register-based conditional execution

## Key Lessons

- Manual PTX predication via inline asm for simple conditional-zero patterns provides no measurable benefit over plain if-else — nvcc already compiles if(c) { x = 0; } into predicated SEL/MOV instructions at SASS level, making hand-written PTX redundant for this pattern
- Reserve inline PTX predication for complex multi-instruction conditional sequences where the compiler may not eliminate the branch; for single-assignment conditional zeroing, the compiler already produces optimal predicated code
