## Challenge

Skeleton kernel lacks sequential initialization logic and global memory write — TODO markers need to be replaced with a loop that sets Array element i to T(i) and writes the result to global memory


## Solution

Implement unrolled loop using CUTLASS_PRAGMA_UNROLL with Array::at(i) accessor to sequentially initialize each element as T(i), then assign the entire Array storage to ptr[threadIdx.x] in a single coalesced write

## Key Lessons

- CUTLASS_PRAGMA_UNROLL on compile-time-bounded loops (for i < N where N is a template parameter) enables the compiler to fully unroll array initialization, eliminating loop overhead and enabling register-level optimization
- Writing the entire Array<T,N> storage in one assignment (ptr[threadIdx.x] = storage) leverages cutlass::Array's contiguous storage layout for a single vectorized global memory store rather than element-by-element writes
