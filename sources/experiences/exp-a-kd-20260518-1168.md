## Solution

8MB to ~2MB, offering moderate spatial locality improvement. However, each query head still reads HEAD_SIZE=64 half-values per block regardless of stride, so the per-call dot-product workload dominates equally in both versions. T

## Key Lessons

- The before code uses an 8-head MHA K-cache layout (block_stride=8192), while the after code uses a 2-head GQA layout (block_stride=2048) with head_mapping={0,0,0,0,1,1,1,1}.
