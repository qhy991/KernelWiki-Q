KernelBench level_1 task_1: Square matrix multiplication (C = A * B, N=2048)

## Key Lessons

- Shared memory tiling reduces global memory accesses by loading tiles cooperatively
- Float4 vectorized loads improve memory bandwidth utilization when 16-byte aligned
- 32x32 thread blocks provide better SM utilization for 2048x2048 matrices
- PyTorch C++ Extension requires torch::empty() for allocation, never cudaMalloc
- Progressive optimization (one change per round) helps isolate performance impact
