## Key Lessons

- Shared memory tiling is effective for reducing global memory bandwidth in GEMM
- TILE_SIZE=32 provides a good balance between shared memory usage and computation
- Boundary checking is essential for matrices that don't divide evenly by tile size
- Correctness verified with max_diff < 1e-4 on N=128 test case
