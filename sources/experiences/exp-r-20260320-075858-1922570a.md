## Key Lessons

- Shared memory tiling provides correct results for matrix multiplication
- TILE_SIZE=32 is a reasonable starting point but may not be optimal for performance
- PyTorch C++ Extension format works correctly with torch::Tensor and torch::empty()
- TORCH_CHECK should be used instead of C10_CUDA_CHECK for error handling
