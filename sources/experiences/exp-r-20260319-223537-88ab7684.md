KernelBench level_1 task_2: Matrix multiplication C = A * B with shapes A(M,K), B(K,N), C(M,N). Default sizes: M=1024, K=4096, N=2048.

## Key Lessons

- Shared memory tiling is not always beneficial - depends on problem size and K dimension
- For K=4096 with M=1024, N=2048, naive kernel outperforms tiled versions
- Synchronization overhead in shared memory kernels can be significant
- Always test optimizations empirically - assumptions may not hold for specific sizes
