## Solution

Use AlignedArray<float, 4> vector loads (16 bytes per LDG transaction) with interleaved addressing so that 8 threads per row each issue 2 vectorized loads covering 8 elements total, achieving coalesced 128-byte memory transactions across the warp

## Key Lessons

- Use vectorized loads (AlignedArray/float4) with interleaved thread addressing to coalesce memory transactions — each thread loads ELTS_PER_LDG elements per transaction, and threads within a group access contiguous memory, producing full 128-byte cache-line transfers
- Calculate vectorization parameters from problem dimensions: VPT = elements per thread, THREADS_PER_ROW = num_experts / VPT, LDG_PER_THREAD = VPT / ELTS_PER_LDG — this determines how to partition work for coalesced access
