## Key Lessons

- Shared memory tiling is effective for matrix multiplication to reduce global memory bandwidth pressure
- TILE_SIZE=16 works well for BMM with m=128, k=256, n=512
- Proper __syncthreads() calls are critical after loading tiles and before computing partial results
- Boundary checks needed when loading tiles to avoid out-of-bounds access
