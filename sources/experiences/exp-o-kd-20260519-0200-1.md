## Solution

Guard the accumulator-buffer copy with an is_accumulator_needed flag (K > 0); when K=0, fill register fragments with zero (the identity element for accumulation) via a simple register-fill loop, bypassing the global-memory load path entirely

## Key Lessons

- Guard accumulator-buffer reads with a K > 0 check in GEMM epilogue code — when no MMA operation produced data (K=0), the accumulator buffer is uninitialized and reading it produces garbage; filling registers with the identity element (zero for sum accumulation) is both correct and avoids unnecessary global-memory loads
- Register-fill with a compile-time-constant (zero) is cheaper than a global-memory load-copy pipeline — the zero path avoids all memory traffic, yielding a modest speedup in addition to correctness
