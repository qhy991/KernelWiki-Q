## Key Lessons

- The after code eliminates the source array read entirely and computes alpha*accum directly via mul_accum, skipping both the source load and the beta*source multiply.
- The before code reads both accumulator (int→float conversion) and source arrays from global memory, converts them to float fragments, computes beta*source (beta=0, yielding zeros), then alpha*accum+0 = alpha*accum, applies relu, and converts back to int via __float2int_rn.
