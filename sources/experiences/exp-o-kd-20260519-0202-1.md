## Challenge

When K=0, the unconditional copy(tiled_t2r, tTR_tAcc_mn, tTR_rAcc) reads from TMEM locations never populated by MMA, loading undefined garbage into the register accumulator fragment which then propagates through epilogue fusion callbacks to global memory output


## Solution

Guard the TMEM-to-register copy with an is_accumulator_needed check: when true the copy proceeds normally, when false (K=0) fill the register fragment with zero (the identity element for accumulation) so downstream epilogue operations produce correct results without invalid TMEM reads

## Key Lessons

- Guard TMEM-to-register accumulator copies with a semantic validity check (e.g. K>0) and zero-fill the register fragment when no valid MMA output exists — zero is the identity for accumulation, so epilogue fusion callbacks (scale, bias, etc.) produce correct results without requiring special-case handling downstream
- Unconditional tensor-memory reads from unpopulated TMEM locations produce undefined values, not zeros — always pair TMEM load paths with a condition that ensures the MMA actually wrote to those locations before attempting a copy
