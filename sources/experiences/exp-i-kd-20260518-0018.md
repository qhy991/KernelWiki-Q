## Challenge

Implement kernel following the CUTLASS_device_function interface pattern with constraints: Target: SM 70+ (Volta and newer); Shared memory limit: 48KB-100KB depending on GPU; Uses CUTLASS CUTE library for tensor operations

## Key Lessons

- Interface pattern 'CUTLASS_device_function': the implementation follows the specified skeleton contract. Key constraints: Target: SM 70+ (Volta and newer); Shared memory limit: 48KB-100KB depending on GPU
- Simple CUDA kernel implementation demonstrates the basic pattern: compute thread index, check bounds, perform work, write result. This pattern applies to any element-wise parallel operation
