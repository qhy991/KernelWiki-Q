## Solution

Add a bool kIsMLA template parameter and wrap all V-cache load/store operations in if constexpr (!kIsMLA), so the compiler completely eliminates V-cache pointer computation, load, and store at compile time for MLA workloads, halving memory traffic with zero branch overhead

## Key Lessons

- Use if constexpr with a template parameter to eliminate unnecessary data transfers at compile time rather than runtime branching — the compiler removes dead code entirely, yielding zero-cost specialization for different model architectures
- For attention variants that merge K and V caches into a single latent (e.g., MLA), V-cache transfers are redundant and can be conditionally compiled out to halve memory traffic per transfer call
