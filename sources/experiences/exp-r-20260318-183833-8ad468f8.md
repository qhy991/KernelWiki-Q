AutoResearch Round 2: Metal FP16 MatMul optimization with tiled kernel

## Key Lessons

- Threadgroup memory tiling is highly effective for matrix multiplication - achieved 2x speedup over naive implementation
- 16x16 tile size works well for 1024x1024 matrices on Metal
- Cooperative loading of tiles by threadgroup reduces global memory bandwidth pressure
- threadgroup_barrier synchronization is critical before using shared memory tiles
- FP32 accumulation maintains precision while FP16 output meets requirements
