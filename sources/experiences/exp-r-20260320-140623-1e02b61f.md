KernelBench level_1 task_19: ReLU activation element-wise operation

## Key Lessons

- Float8 vectorization provides ~2.9% speedup for memory-bound ReLU
- 256 threads/block is optimal for this workload
- Increasing block size to 512 did not improve performance
- Combining float8 with larger block size showed no synergistic benefit
