## Solution

Replace 4 scalar float load/store pairs with a single 128-bit packed load/store via reinterpret_cast<float4*> (LDST128BITS macro), causing nvcc to emit LDG.128/STG.128 instructions and reducing total memory instructions from 8 to 2

## Key Lessons

- Use reinterpret_cast<float4*> to pack 4 contiguous float loads/stores into a single 128-bit vectorized memory transaction (LDG.128/STG.128), reducing memory instruction count by 4x and lowering dispatch pressure — applies whenever accessing 4-aligned float arrays
- Prefer vectorized load/store macros (LDST128BITS pattern) over manual scalar unrolling: the compiler emits wider memory instructions automatically, improving memory throughput without changing thread count or grid configuration
