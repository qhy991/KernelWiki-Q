KernelBench level_1 task_14: Upper triangular matrix multiplication (C = triu(A * B))

## Key Lessons

- Upper triangular matrix multiplication can be implemented as full GEMM + triu mask
- Shared memory tiling (16x16) provides good baseline performance for GEMM
- Separate kernels for GEMM and triu mask improve code clarity
- PyTorch Extension format requires torch::empty() instead of cudaMalloc
