## Challenge

Implement Flash Attention with Q8_0 quantized KV cache. Key challenges: 1) Q8_0 dequantization (scale * int8), 2) Online softmax for numerical stability and single-pass computation, 3) Correct attention formula: output[m] = sum_{seq} softmax(Q @ K[seq]^T / sqrt(d)) * V[seq][m]


## Solution

Load Q vector into shared memory, compute online softmax in single pass over sequence


### Key Changes

- Online softmax: single pass instead of two
- Q vector in shared memory for efficient dot products
- Proper Q8_0 dequantization: scale * int8 value


## Key Lessons

- Online softmax is essential for performance - reduces runtime by 50%
- Q8_0 dequantization requires accessing block scale and int8 values correctly
- Full vector dot product Q @ K[seq]^T is required, not single element
- Shared memory for Q vector reduces global memory access
