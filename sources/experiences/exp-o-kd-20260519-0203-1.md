## Solution

Guard the accumulator release block with is_accumulator_needed so that when K=0 the release and state advance are skipped entirely, preserving pipeline state consistency while allowing normal operation when K>0

## Key Lessons

- Guard pipeline consumer_release with a validity predicate (e.g., is_accumulator_needed) when the kernel may execute with degenerate dimensions like K=0 — skipping release for unconsumed buffers prevents pipeline state corruption without affecting the normal K>0 path
