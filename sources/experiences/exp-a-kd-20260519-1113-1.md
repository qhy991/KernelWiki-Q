## Solution

Fuse AND+OR into a single LOP3 (LOgic Predicate 3-input) PTX instruction via lop3.b32 with look-up table (0xf0 & 0xcc) | 0xaa encoding the (a & b) | c truth table, reducing nibble extraction from 2 instructions to 1 per half2 pair

## Key Lessons

- LOP3.b32 can fuse any two-input bitwise operation (AND+OR, AND+XOR, etc.) into a single hardware instruction on Ampere+ GPUs — apply whenever you combine extracted bit fields with constants via chained bitwise ops
- Instruction savings from replacing 2 ALU ops with 1 LOP3 are negligible when FP16 arithmetic (__hsub2, __hfma2) dominates execution time; the optimization becomes significant only when the fused operation is in the hot path of a large K-dimension loop such as MarlinMoE weight dequantization
