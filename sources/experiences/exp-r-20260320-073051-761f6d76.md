## Key Lessons

- Shared memory is effective when multiple threads access the same data element
- For diagonal matrix multiplication, each row of threads shares the same diagonal element
- Using threadIdx.x == 0 to load shared memory reduces redundant global memory accesses
- Correctness maintained: correct=true, max_diff=0.0
