## Solution

CUDA kernel with shared memory for top-k finding and curand for categorical sampling

## Key Lessons

- For stochastic operations in test frameworks, use the same RNG seed for all batch elements
- curand_init(seed, sequence_offset, offset) - sequence_offset affects the random sequence, must be consistent across batches
- Top-k sampling requires two steps: find top-k indices, then categorical sample from those probabilities
