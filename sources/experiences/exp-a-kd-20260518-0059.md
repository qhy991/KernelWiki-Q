float_only_alpha

## Solution

The optimization's primary value is the compile-time correctness guarantee that source is never read regardless of beta's runtime value, and the host-side memory savings from not needing to allocate a source tensor, rather than measurable kernel-latency improvement.

## Key Lessons

- The after kernel eliminates the source-load conditional branch at compile time via the SourceNeeded template parameter (SourceNeeded=false for OnlyAlphaScaling).
