## Solution

The after kernel replaces this with a shared-memory tiling approach: 192 threads cooperatively load 192 float4 values (768 bytes each) into shared memory in one coalesced 128-bit read per thread, then each thread reads 3 float elements from shared memory, inserts a zero at every 4th position, and...

## Key Lessons

- This results in uncoalesced 4-byte stores and redundant index math for every element including padding zeros.
