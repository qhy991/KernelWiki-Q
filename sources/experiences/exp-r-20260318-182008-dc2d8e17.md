Task 18: Matmul with transposed both (C = A.T * B.T) where A is (K,M), B is (N,K), output is (M,N)

## Key Lessons

- For matrix multiplication, shared memory tiling is essential for performance
- The transposed operation C = A.T * B.T requires careful indexing: C[m][n] = sum_k(A[k][m] * B[n][k])
- TILE_SIZE=16 provides good balance between shared memory usage and parallelism for this problem size
