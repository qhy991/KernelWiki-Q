KernelBench level_1 task_12: Diagonal matrix multiplication C = diag(A) @ B

## Key Lessons

- Shared memory reduces global memory reads for diagonal elements
- 16x16 block size is optimal for this workload
- Block size tuning can degrade performance if not carefully matched to memory access patterns
- Torch native is already highly optimized for simple element-wise operations
