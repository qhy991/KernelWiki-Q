## Solution

Replace scalar float output writes with float4 packed stores via reinterpret_cast, coalescing 4 quantized float values into a single 16-byte write transaction; read shared memory in __half2 pairs and compute 4 quantized values before packing into float4

## Key Lessons

- Use float4 reinterpret_cast stores to quadruple output write throughput when writing contiguous float arrays — coalesces 4 separate 4-byte stores into one 16-byte transaction, applies whenever output alignment permits
- Combine shared memory reads with output vectorization: read 2 __half2 pairs from shared memory, quantize 4 values, then pack into a single float4 store — reduces both shared memory read latency and global store count
