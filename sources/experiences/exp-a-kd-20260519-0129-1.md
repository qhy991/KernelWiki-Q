## Solution

Capture the shared-memory dropout mask once before the hot loop into a single uint32_t register-resident bitmask, then apply it per-iteration using a cheap BITTEST instruction, eliminating all shared-memory loads, 2D index computations, and bounds checks from the hot loop

## Key Lessons

- Hoist invariant shared-memory reads out of hot loops into register-resident bitmask registers — each eliminated smem load saves a shared-memory transaction and associated index arithmetic per iteration
- Compress boolean masks into uint32_t bitmasks when the element count per thread is ≤32 — a single 32-bit register replaces up to 32 shared-memory floats and the per-element test becomes a single BITTEST instruction
- Apply inverse scaling (1/(1-p)) at the kept elements during training dropout to maintain expected magnitudes, and place a __syncthreads() barrier after the one-time smem capture to free shared memory for reuse by subsequent GEMM stages
