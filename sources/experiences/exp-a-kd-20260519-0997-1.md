## Solution

Add a compile-time bool constant kIsMLA and guard the entire V-cache load/store block with if constexpr (!kIsMLA), so the compiler elides all V-cache global memory instructions when MLA mode is active, halving the bytes transferred per work item

## Key Lessons

- Use if constexpr with a compile-time bool to dead-code-eliminate optional data paths in CUDA kernels — the compiler removes the guarded instructions entirely, yielding zero runtime overhead and reduced register pressure
- In bandwidth-bound copy/transfer kernels, eliminating unnecessary data movement (e.g., V-cache when only K-cache is needed for MLA) directly translates to proportional speedup since runtime is dominated by global memory traffic
