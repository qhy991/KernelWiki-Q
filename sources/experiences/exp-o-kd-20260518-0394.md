## Challenge

The before code loads one float at a time from expanded_permuted_rows in the inner k-loop, issuing 4-byte global load instructions per element per k-iteration. With 8 k-iterations over 2048 columns, each thread issues 8*2048=16384 scalar loads


## Solution

The after code reinterprets the float pointer as uint4 (128-bit) and loads 4 floats per instruction, reducing total load instructions by 4x to 8*512=4096.
