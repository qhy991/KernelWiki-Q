## Solution

Remove all outLogits writes and the corresponding logits reads in both loops of the shortcut path, keeping only the outIndices assignments which are the sole required output in this code path

## Key Lessons

- Eliminate redundant global memory reads and writes in shortcut/early-exit paths where only a subset of outputs is required — removing logits reads and outLogits writes in the rowLen <= kTopK path cuts 2 * kTopK global memory transactions per row with no correctness impact
- When a kernel has distinct code paths with different output requirements, audit each path independently to identify outputs that are written but never consumed downstream
