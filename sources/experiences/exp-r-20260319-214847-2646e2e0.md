## Key Lessons

- float4 loads 4 floats (16 bytes) at once, improving memory bandwidth
- Alignment check (address & 3) == 0 ensures safe float4 usage
- Fallback to scalar loads prevents misalignment issues
- Vectorization works well with existing shared memory tiling and loop unrolling
