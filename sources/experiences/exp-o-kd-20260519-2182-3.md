## Solution

Use vec4_t_f (4-wide aligned float struct) to load and process 4 input elements per global memory transaction, with a tail loop for remaining elements when num_elems is not a multiple of 4 — applied to both absmax computation (thread_max_vec) and quantization (scaled_fp8_conversion_vec)

## Key Lessons

- Use vec4_t (128-bit aligned struct of 4 floats) to coalesce 4 scalar loads into a single 128-bit global memory transaction, effectively quadrupling memory throughput for bandwidth-bound kernels — requires input to be 16-byte aligned and a tail loop for remainder elements
- Apply the same vectorized load pattern to both the reduction pass and the transformation pass for consistent memory throughput gains across the entire fused kernel
