## Key Lessons

- Lower triangular matmul has inherent performance limitations due to sparse computation pattern
- Shared memory tiling provides minimal benefit when triangular structure causes thread divergence
- cuBLAS general matmul optimization outperforms custom triangular kernels for moderate sizes
- For triangular matmul, consider using cuBLAS with post-processing (tril) instead of custom kernel
