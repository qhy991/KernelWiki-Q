## Key Lessons

- Early return in CUDA kernels can break shared memory synchronization
- All threads in a block must participate in shared memory operations
- Upper triangular masking should only apply to output writes, not computation
- 32x32 tile size gave best performance for this GEMM kernel
- PyTorch cuBLAS is highly optimized and difficult to beat for matrix multiplication
