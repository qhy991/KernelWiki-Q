## Solution

Guard the entire V-cache transfer block with if constexpr (!kIsMLA), a compile-time conditional that eliminates V-cache load_vec/store_vec, src_v/dst_v pointer arithmetic, and vec_v storage allocation when MLA mode is active, halving global memory traffic with zero runtime overhead

## Key Lessons

- Use if constexpr with a compile-time flag (e.g. kIsMLA) to conditionally eliminate entire code blocks at compile time — the dead branch is not instantiated, removing memory transfers, pointer arithmetic, and local storage with zero runtime cost
- When a kernel handles multiple data streams that may not all be needed in every configuration, guard each stream with its own compile-time flag to allow the compiler to produce specialized kernels that avoid unnecessary memory traffic
