## Solution

Extract the inline if constexpr branch into a standalone __device__ __forceinline__ cast_to_float2 helper that dispatches on packed_t via the same compile-time mechanism, producing identical PTX (cvt.rn.f32.f16x2) while enabling reuse across kernels

## Key Lessons

- Extracting if constexpr type dispatch into a __device__ __forceinline__ helper produces identical PTX and zero runtime overhead because if constexpr resolves at compile time — apply this refactoring freely to improve readability and enable cross-kernel reuse without performance concern
- Adding a float2 pass-through case in the cast helper (return float2(val)) makes the utility generic for mixed-precision pipelines where the input may already be in the target type
