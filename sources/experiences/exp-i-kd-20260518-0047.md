## Challenge

Implement kernel following the CUTLASS_kernel_template interface pattern with constraints: Target: SM 75+ (Ampere and later); Shared memory access should use 128-bit LDSMI; count = sizeof(T) / 4 determines elements per thread

## Key Lessons

- Interface pattern 'CUTLASS_kernel_template': the implementation follows the specified skeleton contract. Key constraints: Target: SM 75+ (Ampere and later); Shared memory access should use 128-bit LDSMI
- Simple CUDA kernel implementation demonstrates the basic pattern: compute thread index, check bounds, perform work, write result. This pattern applies to any element-wise parallel operation
