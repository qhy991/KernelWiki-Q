Task 8: Square matrix multiplication with irregular shapes (M=8205, K=2949, N=5921)

## Key Lessons

- Use 32x32 tile size for good balance between shared memory usage and occupancy
- Always implement boundary checks for irregular/non-tile-aligned dimensions
- AT_DISPATCH_FLOATING_TYPES provides type flexibility for PyTorch extensions
- TORCH_CHECK is preferred over C10_CUDA_CHECK for error handling
