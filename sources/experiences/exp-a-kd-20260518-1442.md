batch_32tokens_static

## Solution

GPU amortizes across all 32 blocks. The before version passes scales via the kernel launch parameter buffer (constant memory). The marginal 6% improvement falls within microbenchmark noise: the main loop iterates 1024 times per bloc

## Key Lessons

- The after kernel replaces scalar-by-value k_scale/v_scale with pointer dereferences (*k_scale, *v_scale) read from GPU global memory.
