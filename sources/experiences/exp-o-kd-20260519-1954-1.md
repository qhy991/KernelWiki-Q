## Solution

Replace scalar per-element half loads with 128-bit aligned packed loads using array_t<half,8> with __align__(16), forcing the compiler to emit ld.128/st.128 instructions that transfer 8 half values per memory operation, paired with unrolled upcast/downcast helpers that process all 8 elements without per-element conversion call overhead

## Key Lessons

- Use __align__(16) array_t<T,16/sizeof(T)> packed types to force ld.128/st.128 instructions instead of scalar half loads — the 8x reduction in memory instructions is especially impactful for fp16 where each scalar load transfers only 2 bytes
- Wrap half-to-float and float-to-half conversions in unrolled packed helpers (upcast_packed/downcast_packed) to let the compiler pipeline all 8 conversions without per-element function call overhead — applies whenever converting between half and float in bandwidth-bound kernels
