## Key Lessons

- TILE_SIZE=16 provides good balance between shared memory usage and parallelism for N=2048
- Boundary checks (row < N, col < N) are necessary for non-multiple tile sizes
- __syncthreads() must be called after loading shared memory tiles
- TORCH_CHECK(cudaGetLastError() == cudaSuccess, ...) is the correct error handling pattern
