## Challenge

Implement optimized RMS norm kernel for llama2_7b with hidden_size=128


## Solution

CUDA kernel with warp-level reduction for efficient RMS computation


### Key Changes

- Used __shfl_down_sync for efficient warp reduction
- Shared memory for block-level aggregation
- One thread per element in hidden dimension


## Key Lessons

- RMS norm kernel can be efficiently implemented with one thread per hidden dimension element
- Warp reduction using __shfl_down_sync is efficient for summing across threads
- Avoid duplicate variable declarations in CUDA kernel parameters
