## Solution

Fuse fp8 load and dequant into a single triple-nested loop using a stack-local fp8_temp[8] temporary per chunk, eliminating the intermediate Vlocalb8 buffer entirely and reducing peak register usage by the size of the removed staging buffer

## Key Lessons

- Fusing load and dequant into a single loop pass eliminates an intermediate staging buffer and reduces peak register pressure — applies when the intermediate buffer is consumed only once in a subsequent pass
- At small head sizes (HEAD_SIZE=64, VHELOOP=1) the compiler may already fit both buffers in available registers, making the deferred-vs-inline distinction neutral; the optimization becomes more impactful at larger head sizes where the staging buffer exceeds register capacity and spills to local memory
