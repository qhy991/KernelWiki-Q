Implement attention operator using Metal on macOS with kernel.metal and metal_test.mm

## Key Lessons

- Metal kernel dispatch uses MTLSize with (x,y,z) dimensions matching thread_position_in_grid
- Always pass dimensions as constant buffers (e.g., seq_len, head_dim, num_heads)
- Bounds checking with if (gid.x < num_heads && gid.y < seq_len ...) is essential
- Use MTLResourceStorageModeShared for buffers that need CPU access after GPU computation
- Softmax requires two-pass: find max, then compute exp and sum, then normalize
