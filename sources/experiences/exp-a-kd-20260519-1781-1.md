## Challenge

Standard GEMM dispatch for small-M tiles (M=16, N=4096, K=4096) uses a generic kernel path that does not leverage CUTLASS swap-AB kernel variants optimized for tall-skinny matrix shapes on SM90 hardware


## Solution

Swap-AB transformation reformulates C=A*B as D=B^T*A^T, combined with dimension-aware dispatch that splits M<=64 into M<=16 and M in (16,64] with N-threshold sub-branching (N<=1280), triggering specialized CUTLASS swap-AB kernel variants on SM90

## Key Lessons

- Swap-AB reformulation (C=A*B → D=B^T*A^T) triggers different CUTLASS kernel variants on SM90 better suited for small-M tiles — applies when M<=64 and the dispatch framework supports swap-AB paths with scale-argument reversal
- Dimension-aware dispatch gains are hardware-specific: the same swap-AB logic has no effect on SM80 where PyTorch falls back to cuBLAS for all shapes — verify target architecture before applying
