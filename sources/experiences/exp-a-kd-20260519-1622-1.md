## Solution

Introduce cp.async.cg.shared.global PTX instructions to initiate asynchronous 16-byte global-to-shared memory copies with a 3-stage shared memory buffer, so each compute iteration reads from shared memory while the next iteration's global load is already in flight, overlapping async copies with compute to hide global memory latency

## Key Lessons

- Use cp.async.cg.shared.global with multi-stage shared memory buffers to overlap global memory loads with compute — the prologue pre-loads N-1 stages, then each iteration waits for the oldest stage while launching the next async copy, hiding global memory latency without additional threads
- cp.async.commit_group + cp.async.wait_group<N> provides lightweight synchronization for async copies without full __syncthreads() on every issue — wait_group<N> where N is the number of outstanding groups to tolerate
