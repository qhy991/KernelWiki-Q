## Solution

Remove the outer seqlen-modulo guard and replace chunk-index multiplication (chunk * kChunkSize) with a cumulative tokens_processed counter; compute chunk_tokens = min(chunk_size, seqlen - tokens_processed) per chunk and apply the boundary check unconditionally, simplifying control flow and improving correctness for variable-length chunk tails

## Key Lessons

- Removing a redundant outer modulo guard (seqlen % N != 0) and applying the boundary check unconditionally eliminates a branch that is almost always taken, simplifying the control flow without measurable cost — applies whenever the guard condition is true for the common case
- Prefer cumulative token counters (tokens_processed) over chunk-index multiplication (chunk * chunk_size) for chunk iteration — the additive form avoids redundant multiplication and naturally handles variable-length tail chunks
