## Challenge

The before code uses convert_to_uint32_v2 with top-byte extraction ((fp32_bits >> 24) & 0xFF) for the Pass 0 coarse histogram over all 8192 elements. This involves a 32-bit sign-flip and a shift-mask per element


## Solution

The after code uses convert_to_uint8 which converts FP32 to FP16 via __float2half_rn, then extracts the high byte with a 16-bit sign-flip.
