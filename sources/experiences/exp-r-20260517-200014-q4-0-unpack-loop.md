## Key Lessons

- Q4_0 block = 18 bytes = 32 values. 2 bytes FP16 scale + 16 bytes quantized data.
- Unpack ALL 32 values from each block. Never skip elements with conditional logic like k_in_block == 0.
- Extract nibbles: high = byte >> 4, low = byte & 0x0F. Process both for each of the 16 data bytes.
- Sign extend 4-bit to 8-bit: if nibble >= 8, val = nibble - 16; else val = nibble.
- Scale is FP16, stored in first 2 bytes. Convert to float before multiplying with unpacked values.
- The outer loop advances by 18 bytes per Q4_0 block, not 32 (because 32 values are packed into 16 bytes + 2 scale bytes).
- For correctness testing, verify the unpacked values against a reference implementation before integrating into the kernel.
