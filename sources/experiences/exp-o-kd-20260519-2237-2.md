## Solution

Add an is_a_8bit flag to the cache size calculation, using 1 byte per element for INT8 activations instead of the hardcoded 2 bytes, halving A-matrix shared memory consumption

## Key Lessons

- Size shared memory buffers based on actual element size (INT8=1 byte, FP16=2 bytes) rather than using a conservative maximum — the A-matrix buffer alone can be halved for INT8 activations, which is critical for fitting quantized GEMM kernels within shared memory limits on constrained architectures
