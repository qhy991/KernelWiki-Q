## Solution

Replace 32 per-element bit extractions with 4 byte-level reads: iterate over the underlying byte array and check if any byte equals 0x00 (meaning all 8 packed bits are false), exploiting the AND identity that a single zero byte implies the entire reduction result is false

## Key Lessons

- For packed sub-byte data types, operate on the underlying byte/container instead of extracting individual elements via bit-shift/mask — this reduces loop iterations by 8x and eliminates per-element extraction overhead, applicable whenever operations on packed bits have a byte-level algebraic shortcut
- Exploiting the identity property of AND (any 0x00 byte means the entire group of 8 bits is false) converts 8 scalar comparisons into one byte-level zero-check, a pattern generalizable to any reduction over packed boolean data
