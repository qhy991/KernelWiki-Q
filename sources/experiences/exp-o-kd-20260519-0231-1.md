## Solution

Coarsen the outer loop to process FPP=2 fragments per iteration, staging all FPP fragments into doubled shared memory between barrier pairs, halving total barriers from 2*kIterations to 2*(kIterations/FPP) at the cost of 2x SMEM usage

## Key Lessons

- Loop coarsening that batches multiple fragments between a single barrier pair reduces __syncthreads() count proportionally to the coarsening factor — applies whenever SMEM capacity can absorb the extra fragments and the per-fragment work is lightweight relative to barrier cost
- Trading 2x shared memory for 2x fewer barriers is profitable on modern GPUs (A800) where SMEM per block is abundant (up to 48–164 KB) and barrier latency is ~30 cycles per warp, making barrier elimination the dominant win for short per-fragment workloads
