Task 9: Tall skinny matrix multiplication C = A * B where A is (16384, 16) and B is (16, 16384)

## Key Lessons

- For tall skinny matrices with small inner dimension, shared memory tiling is highly effective
- 16x16 thread blocks provide good occupancy for this problem size
- Single iteration loading works well when K matches tile size
- TORCH_CHECK is required for CUDA error checking in PyTorch extensions
