KernelBench level_1 task_18: C = A.T * B.T matrix multiplication. A(K,M), B(N,K), output C(M,N)

## Key Lessons

- Transposed matmul requires careful indexing: A[k][m] and B[n][k] for C[m][n]
- Naive implementation provides correct baseline for optimization
- 16x16 thread blocks work well for M=1024, N=2048, K=4096 sizes
