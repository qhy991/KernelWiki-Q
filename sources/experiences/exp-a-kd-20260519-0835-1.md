## Solution

Bundle all kernel parameters into a single FusedMetadataCopyParams struct and pass it via __grid_constant__, which routes the entire struct through the constant memory cache so all threads access scalar fields (bs, max_len, stride) and pointer fields through the faster constant cache path instead of global memory

## Key Lessons

- Use __grid_constant__ to pass a bundled parameter struct instead of many individual kernel arguments — constant cache provides a dedicated low-latency path for read-only parameter access shared across all threads in the grid
- Parameter passing overhead is proportionally significant for small kernels (few threads, little work per thread) — restructuring parameter delivery has higher relative impact when the compute-to-parameter-access ratio is low
