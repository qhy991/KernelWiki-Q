## Solution

Replace manual LDST128BITS/HALF2 reinterpret_cast macros with CuTe tensor abstractions: make_tensor + local_tile for per-thread memory partitioning, copy() for vectorized load/store (generating identical LDG.128/STG.128), and recast<half2>() for type-level packing of arithmetic operations

## Key Lessons

- CuTe copy()/recast<> abstractions generate identical LDG.128/STG.128 and half2 packed arithmetic instructions as manual reinterpret_cast macros, yielding equivalent performance for bandwidth-bound elementwise kernels — prefer CuTe when portability and composability with higher-dimensional layouts are needed
- CuTe local_tile() declaratively partitions global memory into per-thread tiles, replacing manual stride calculations like 8 * (blockIdx.x * blockDim.x + threadIdx.x) — reduces indexing errors and scales naturally to multi-dimensional tiling patterns
