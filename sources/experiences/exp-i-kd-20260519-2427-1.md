## Challenge

Implement a CUDA kernel that performs element-wise addition of a scalar uint128 base value to each element of a uint128 input array, using a 1D grid with coalesced memory access and proper bounds checking


## Solution

Use standard 1D thread indexing (threadIdx.x + blockIdx.x * blockDim.x) with a bounds check (tid < N) to map each thread to one array element, then perform a single coalesced load-add-store: output[tid] = input[tid] + base

## Key Lessons

- For element-wise operations on contiguous arrays, the canonical 1D grid pattern (tid = threadIdx.x + blockIdx.x * blockDim.x; if (tid < N)) produces naturally coalesced memory accesses when consecutive threads read/write consecutive array indices
- cutlass::uint128_t supports operator+ natively, allowing element-wise 128-bit integer addition without manual carry propagation between low/high 64-bit halves
