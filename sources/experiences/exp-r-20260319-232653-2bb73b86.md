KernelBench level_1 task_3: Batched Matrix Multiplication (BMM). Shapes: A(batch, m, k), B(batch, k, n), C(batch, m, n). Default: batch=128, m=128, k=256, n=512.

## Key Lessons

- Must use #include <torch/extension.h>
- Use torch::Tensor for wrapper parameters, not float*
- Use torch::empty_like() for output allocation
- Include PYBIND11_MODULE with m.def for forward function
- Naive BMM is correct but slow - needs optimization for speedup
