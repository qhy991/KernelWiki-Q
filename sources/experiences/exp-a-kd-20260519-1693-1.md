## Solution

Replace the branching bubble-sort with a fixed 5-swap branchless sorting network for N=4 using min/max intrinsics (TOPK_SWAP macro), eliminating all conditional branches by computing both orderings and assigning without predicates

## Key Lessons

- For small fixed-size sorts (N=4), the CUDA compiler converts simple if-swap patterns to predicated moves, so branchless min/max sorting networks do not yield measurable speedups — the optimization is stylistic rather than performance-critical at these sizes
- A fixed sorting network for N=4 requires exactly 5 compare-swap operations using the known-optimal Batcher odd-even merge sequence, replacing a variable-iteration bubble sort with a fully unrolled deterministic sequence
