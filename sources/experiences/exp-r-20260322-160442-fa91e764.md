## Key Lessons

- NEVER add cudaDeviceSynchronize() in extern C wrapper - causes 2.5x slowdown
- Always strip PYBIND11_MODULE and torch headers before testing with unified_test_runner.py
- For small hidden sizes (128), single-warp-per-row approach is optimal
- Some baselines are already highly optimized - just need proper stripping
- RN_03 (hs=512) baseline achieved 255.6x ratio without any code changes beyond stripping
