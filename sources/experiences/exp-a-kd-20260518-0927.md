## Key Lessons

- This transposition forces cuBLAS to traverse both input matrices with non-contiguous stride access patterns—the B matrix (4096,4096) is accessed with stride k=4096 and the A matrix (3,4096) with stride m=3, causing uncoalesced global memory loads on SM80.
