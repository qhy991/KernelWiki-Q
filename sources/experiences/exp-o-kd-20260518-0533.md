## Key Lessons

- This 2x reduction in memory instruction count allows the GPU to coalesce the accesses into fewer memory transactions and reduces the instruction overhead in the inner loop.
- The measured 7.8x speedup is amplified by the single-thread launch configuration where kernel launch overhead and loop instruction count dominate; in the real Marlin kernel with full thread blocks, the relative improvement would be smaller but the same vectorization principle applies.
