## Solution

Replace the serial prefix-sum loop with CUB BlockScan ExclusiveSum, which computes the exclusive sum collaboratively across all 1024 block threads using an efficient parallel scan algorithm, distributing the work that was previously done by a single thread

## Key Lessons

- CUB BlockScan ExclusiveSum replaces serial prefix-sum loops with a parallel scan that engages all block threads — particularly effective when the serial loop ran on a single thread while others waited at a barrier, as the parallel scan turns idle threads into useful compute
- Each thread contributes one element to BlockScan (aligned_count for its assigned expert, or 0 for padding threads), making the scan data-parallel rather than control-parallel — this pattern generalizes to any per-thread reduction that needs a prefix sum
