Task 16: C = A.T * B matrix multiplication. A shape (K,M), B shape (K,N), output C shape (M,N). Default sizes: M=1024, K=4096, N=2048.

## Key Lessons

- For C = A.T * B where A is (K,M) and B is (K,N), output is (M,N)
- Naive kernel with 16x16 blocks passes correctness tests
- PyTorch C++ Extension format requires #include <torch/extension.h> and PYBIND11_MODULE
- Use torch::empty() for output allocation, never cudaMalloc/cudaFree
