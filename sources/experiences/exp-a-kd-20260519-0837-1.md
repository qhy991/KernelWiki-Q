## Solution

Fuse 3 separate single-destination kernel launches into one multi-destination kernel that reads each source element once into a register (int32_t val) and writes it to all 3 destination buffers, eliminating 2 kernel launches and 2/3 of redundant source reads while reducing L2 cache pressure

## Key Lessons

- Fuse multiple identical kernel launches that read the same source into a single multi-destination kernel — eliminates redundant global memory reads and N-1 kernel launch overheads, especially impactful for small batches where per-launch overhead (~5us) dominates total execution time
- Read source data into a register variable before writing to multiple destinations to ensure the value is fetched from global memory only once, reducing L2 cache pressure when the same source must be broadcast to several output buffers
