## Challenge

Reduce global memory traffic by caching weight in shared memory


## Solution

Shared memory weight caching with cooperative load


### Key Changes

- Added __shared__ float shared_weight[128]
- First warp loads weight cooperatively via float4
- All warps read from shared memory instead of global


## Key Lessons

- Shared memory caching provides marginal benefit (0.6%) for small hidden_size=128
- Weight is already cached in L1 via __ldg(), so shared memory adds limited value
- For small tensors, overhead of __syncthreads() may offset benefits
