## Solution

Replace the bool mask array with a uint8_t bit-packed mask (1 byte per token instead of 8), eliminating 7 of 8 initialization writes and removing the cumulative offset loop and mode branching by using a direct (bid * draft_token_num + tid) * num_bytes_per_item stride formula

## Key Lessons

- Bit-pack sparse boolean masks into uint8_t (one bit per flag) to reduce memory traffic by 8x — applies whenever each element stores only 0/1 and the mask is read/written in a bandwidth-bound pattern
- Replace cumulative-offset loops that iterate up to O(blockIdx.x) with direct stride formulas (blockIdx.x * stride + threadIdx.x) to eliminate data-dependent iteration counts and ensure uniform execution time across blocks
- Eliminate runtime mode branching (FULL_MASK vs QLEN_ONLY) by designing a single index formula parameterized by a compact stride (num_bytes_per_item), removing divergent warps and simplifying the kernel interface
