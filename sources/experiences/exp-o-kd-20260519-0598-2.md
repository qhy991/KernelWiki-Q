## Solution

Hoist k_rope read outside the head loop into a register buffer (KRopeBufType k_rope_buf), reading once and reusing the value across all 16 head iterations, eliminating 15 redundant global memory loads per lane

## Key Lessons

- When data is shared across loop iterations (e.g., k_rope identical for all heads in a token), hoist the global read outside the loop into a register buffer — eliminates (N-1) redundant global loads where N is the iteration count
- Separate read and write phases (buffer all inputs first, then write all outputs) to decouple memory access patterns from loop structure, enabling both register reuse and independent optimization of each phase
