## Solution

The optimization is most impactful for narrow data types (INT4) in mixed-precision GEMMs where the thread-value layout of the wider MMA type (FP8) forces scattered 4-bit element access. Measured impact requires point-level validation.

## Key Lessons

- Since INT4 elements occupy only 4 bits, a K-major layout forces 16-bit (sub-bank) shared memory loads rather than the 64-bit vectorized loads that maximize shared memory throughput.
- This results in extra load instructions on the critical path and suboptimal utilization of the shared memory bandwidth.
