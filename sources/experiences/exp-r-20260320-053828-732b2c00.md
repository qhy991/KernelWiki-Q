KernelBench level_1 task_10: Batched matrix multiplication (N=16, M=1024, K=2048, L=768) x (K, L) -> (N, M, L)

## Key Lessons

- Batched matmul can be implemented with shared memory tiling similar to standard GEMM
- 16x16 tile size works well for M=1024, K=2048, L=768 dimensions
- PyTorch C++ Extension requires torch::Tensor parameters and torch::empty() for memory allocation
