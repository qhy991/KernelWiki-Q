## Key Lessons

- fp16 requires careful conversion between host and device code
- Accumulating in float then converting to fp16 at the end improves accuracy
- fp16 operations have inherent precision limitations requiring relaxed tolerance
- Metal's half type should be used in shaders, uint16_t in host code
