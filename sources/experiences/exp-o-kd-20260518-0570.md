scalar_fp8_to_half

## Solution

The after code fuses both operations into a single kernel (fp8_to_half_scaled_kernel) that reads each fp8 element, converts to half, promotes to float for the scale multiply, and converts back to half in one pass—eliminating the intermediate buffer allocation, the extra global memory write+read, ...

## Key Lessons

- The before code uses two separate kernel launches: fp8_to_half_unscaled_kernel writes intermediate half values to a temporary buffer d_half_tmp, then scale_half_kernel reads that buffer, applies the scale in float, and writes the final result to d_half_out.
