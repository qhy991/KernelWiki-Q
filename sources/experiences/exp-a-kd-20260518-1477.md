## Key Lessons

- The 256-byte table fits entirely in L2, so each warp's divergent reads are serviced by parallel L2 cache banks rather than sequential constant-cache broadcasts.
- In contrast, the global-memory __device__ LUT (lut_global) benefits from the L2 cache, which handles random access patterns within a warp independently without serialization.
