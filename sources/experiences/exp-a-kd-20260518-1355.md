fp8_to_half_scaled

## Solution

hich does __nv_cvt_fp8_to_halfraw, half_to_float, multiply by scale, and float_to_half in a single function. Both paths use __forceinline__ PTX-assembly helpers that compile to the same cvt.f16.fp8, cvt.f32.f16, mul.f32, cvt.rn.f16.f32 ins

## Key Lessons

- The before code calls a separate vec_conversion_fp8_to_half function for the unscaled fp8-to-half step, then applies float_to_half(half_to_float(h) * scale) in the caller.
