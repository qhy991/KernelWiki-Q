## Challenge

Optimization focus: Add lda parameter to the Marlin kernel and replace all prob_k-based A pointer advances and stride computations with lda.
Problem before: The Marlin GEMM kernel computed the global memory stride of A as prob_k / 8 and used prob_k to advance the A pointer between thread-block slice


## Solution

## Key Lessons
