Metal FP16 matrix multiplication kernel on macOS with runtime compilation

## Key Lessons

- Threadgroup tiling is critical for memory bandwidth - 7.4x speedup
- Vectorization with half2 provides additional 1.58x speedup
- Threadgroup memory indexing must match vectorization pattern (Bsub[k][tid.y*VEC_WIDTH])
- Manual FP16 conversion needed for host code on macOS clang
- Runtime compilation via newLibraryWithSource works well for Metal kernels
