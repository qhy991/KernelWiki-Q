## Key Lessons

- The after kernel regresses on A800 (sm_80) because its shared memory layout is counterproductive on this architecture.
- On SM100 hardware, the transposed layout (LayoutB_T for the first operand) aligns with the Blackwell tensor core's preferred column-major fragment load pattern, providing a net win despite the larger tile.
