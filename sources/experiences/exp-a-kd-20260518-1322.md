## Key Lessons

- The reduction from two 128-bit memory transactions to one per work item eliminates redundant memory instructions and improves memory throughput, producing the measured ~7% latency reduction.
- The key mechanism is that deriving alignment from element bit width ensures the vector width always matches 128-bit access granularity.
