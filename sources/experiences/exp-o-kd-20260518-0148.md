sgemm_1024x1024x1024

## Solution

CUDA 12.8, cuBLAS already auto-dispatches SGEMM to tensor cores under CUBLAS_DEFAULT_MATH for the 1024x1024 shape, so the explicit TF32 math mode only provides a modest 1.15x improv

## Key Lessons

- The after code uses CUBLAS_TF32_TENSOR_OP_MATH and CUBLAS_GEMM_DEFAULT_TENSOR_OP, while the before code uses CUBLAS_DEFAULT_MATH and CUBLAS_GEMM_DEFAULT.
