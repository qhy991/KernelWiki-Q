## Key Lessons

- Critical to convert float to half (uint16_t) before creating Metal buffers when kernel expects fp16
- Use union-based conversion for float/half compatibility in Metal test harness
- fp16 precision has ~3-4 decimal digits accuracy, so tolerance of 0.1 is acceptable for matrix multiplication
- Always bounds-check with 'if (gid < n)' to prevent out-of-bounds writes
- Use dispatchThreads for dispatching in Metal
