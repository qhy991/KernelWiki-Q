## Solution

Use inline PTX cache-streaming hints (ld.global.cs / st.global.cs) on all loads and stores, which tells the GPU to stream data through without allocating L2 cache lines on reads, reducing cache management overhead that is proportionally visible when the kernel is below memory saturation

## Key Lessons

- Use ld.global.cs / st.global.cs PTX cache-streaming hints for copy kernels where loaded data is immediately stored and never reused — the .cs hint bypasses L2 allocation on reads, preventing cache pollution that is proportionally costly when the kernel is below memory saturation and cache management overhead becomes a visible fraction of total latency
