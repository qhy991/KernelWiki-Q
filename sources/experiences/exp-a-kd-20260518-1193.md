## Key Lessons

- The inline approach avoids the large Vlocalb8[][][] allocation, reducing peak register pressure since both buffers would coexist during Phase 2 in the deferred approach.
- For V, which has a 2D buffer structure [VHELOOP][VTLOOP*VBLOCKS], the deferred pattern requires significantly more register space than K's compact 1D Klocalb8[KHELOOP] buffer.
