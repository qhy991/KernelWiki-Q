## Challenge

The before code uses CausalMask<false> (IsQBegin=false), which computes max_blocks_q = ceil_div((blk_coord_q+1)*tile_q + offset_q, tile_k) where offset_q = N_K - N_Q = 4096 - 256 = 3840. This large offset forces every Q-row to visit nearly all 64 K-blocks (total 250 trips across 4 Q-blocks)


## Solution

The after code uses CausalMask<true> (IsQBegin=true), which computes max_blocks_q = ceil_div((blk_coord_q+1)*tile_q, tile_k) without the offset_q term.
