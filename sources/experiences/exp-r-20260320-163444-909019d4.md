## Key Lessons

- Basic shared memory tiling is sufficient for correctness
- Simple optimizations (tile size, vectorization) show minimal improvement
- Register tiling requires careful indexing - easy to introduce bugs
- PyTorch native matmul (cuBLAS) is hard to beat with simple kernels
- For production performance, use cuBLAS directly or CUTLASS library
