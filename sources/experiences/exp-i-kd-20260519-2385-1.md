## Challenge

Skeleton code requires implementing a __device__ function that returns a constant value and is callable from a __global__ kernel, demonstrating basic CUDA device-to-global function call convention


## Solution

Implement the device function to return -1 directly and call it from the global kernel, wrapping the code with a header guard (#ifndef/#define) and #pragma once for safe inclusion

## Key Lessons

- __device__ functions are directly callable from __global__ kernels without any special qualification — CUDA's execution model allows this by default, enabling code reuse across device-side call sites
