## Key Lessons

- Upper triangular matmul optimization: skip lower triangle computation entirely
- For C[i][j] in upper triangle, dot product only needs k from i to j
- Shared memory tiling works well for N=4096 with TILE_SIZE=32
- Loading only non-zero elements into shared memory reduces memory bandwidth
