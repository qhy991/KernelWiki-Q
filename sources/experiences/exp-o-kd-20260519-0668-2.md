## Solution

Replace scalar int32_t stores with 4-wide vectorized stores using an alignas(16) AlignedArray<int32_t, 4> wrapper and reinterpret_cast<Vec*> on the output pointer, reducing the fill's memory transaction count by approximately 4x

## Key Lessons

- Use alignas-aligned struct wrappers (AlignedArray<T, N>) with reinterpret_cast to emit vectorized stores without intrinsics — the compiler auto-generates wide STG instructions when the pointer type has sufficient alignment, reducing memory transaction count by N× for fill and pattern-write kernels
