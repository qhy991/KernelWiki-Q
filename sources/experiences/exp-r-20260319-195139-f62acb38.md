## Key Lessons

- Shared memory tiling (16x16) provides good balance between occupancy and memory reuse
- Coalesced memory access is critical for GEMM performance
- B transposed means accessing B in column-major order - need careful indexing
- PyTorch C++ Extension requires #include <torch/extension.h> and PYBIND11_MODULE
