## Solution

Preload weight values into thread-private registers once before the multi-head loop, then reuse them across all heads, eliminating redundant global memory accesses for the weight tensors

## Key Lessons

- Preload small, repeatedly-used weight tensors into registers before a multi-iteration loop to eliminate redundant global memory traffic — applies whenever the same weights are reused across loop iterations and fit in available registers (e.g., 4 floats per thread for HEAD_DIM=128)
