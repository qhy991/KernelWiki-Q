fp32_rms_norm_llama2_7b_hs512

## Solution

Shared memory reduction for block-level summation

## Key Lessons

- Warp reduction only works within 32 threads - use shared memory for larger blocks
- Shared memory reduction requires __syncthreads() after each step
- RMS norm formula: output = input / sqrt(mean(input^2) + epsilon) * weight
- Strided loops efficiently handle dimensions larger than thread count
