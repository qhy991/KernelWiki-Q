## Key Lessons

- Loop unrolling is a safe optimization that reduces loop overhead without complex register management
- Register tiling can be error-prone and cause compilation issues if not carefully implemented
- Starting with simpler optimizations (loop unrolling) before complex ones (register tiling) is a good strategy
