renorm_topk2_8exp_fallback

## Solution

e before approach writes unnormalized weights to global memory via the topk kernel, then a separate renormalize kernel reads those weights back (8192*2*4=64KB), computes row-wise sums, and writes normalized values back (64KB).

## Key Lessons

- The fused approach eliminates this extra global memory round-trip and one kernel launch.
