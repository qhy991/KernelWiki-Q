## Key Lessons

- For 4D tensor-matrix multiplication, treat (b,i,j) as batch dimensions and compute vector-matrix product for each position
- Shared memory tiling is effective when the same matrix (B) is reused across multiple output positions
- Use coalesced loads for B matrix into shared memory to maximize memory bandwidth
- #pragma unroll helps compiler optimize inner loops for better performance
