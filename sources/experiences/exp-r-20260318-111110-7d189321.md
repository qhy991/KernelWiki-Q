## Key Lessons

- Use thread_position_in_grid for 1D/2D indexing in Metal kernels
- Always bounds-check with gid < n to prevent out-of-bounds writes
- Pass matrix dimensions as constant buffers: constant uint& M [[buffer(3)]]
- Use MTLResourceStorageModeShared for buffers on macOS
- Accept kernel.metal path via argv[1] with fallback to 'kernel.metal'
- Dispatch with appropriate threadgroup size (e.g., 16x16 for matmul)
- Compute CPU reference and compare output for correctness verification
