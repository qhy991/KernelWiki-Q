## Key Lessons

- PyTorch C++ Extension requires #include <torch/extension.h> and PYBIND11_MODULE
- Use torch::empty() for output allocation, never cudaMalloc
- Shared memory tiling with TILE_SIZE=16 works for correctness
- TORCH_CHECK for error handling instead of C10_CUDA_CHECK
