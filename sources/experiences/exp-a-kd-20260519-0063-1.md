## Solution

When is_source_needed() returns false at compile time (beta=0, OnlyAlphaScaling path), eliminate the source tensor load entirely and replace the three-step mul_source+mul_add_accum chain with a single mul_accum(alpha, accum) operation, halving global memory reads from 512KB to 256KB

## Key Lessons

- Eliminate unnecessary global memory loads at compile time when scaling factors make certain inputs unused (beta=0 → source not needed) — halves memory traffic — applies to any epilogue or fused kernel with conditional data dependencies
- Microbenchmarks with small working sets (64K floats = 256KB, fits in L2) may show neutral results even when the optimization is correct, because kernel launch overhead (~0.1ms) and L2 caching mask the bandwidth savings visible only at DRAM-bound scales
