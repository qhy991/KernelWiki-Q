## Challenge

For THREADS_PER_SUBWARP=16, all 4 shuffle-XOR steps are needed for a complete max reduction across 16 threads. The if constexpr optimization does not


## Solution

eliminate any steps; both before and after execute the same number of shuffles. The only difference is the mask: 0xffffffff vs partitioned half-warp

## Results

- **speedup**: 1.000x (neutral)


## Key Lessons

- Both masks achieve correct reduction for 16-thread subwarps, so no performance difference is expected, and measured results should be neutral.
