## Solution

Add a conditional guard is_accumulator_needed = K > 0 that branches between the normal accumulator copy path and a register zero-fill path, ensuring correct output (zeros as the accumulation identity) when no MMA data exists and avoiding unnecessary global memory reads

## Key Lessons

- Guard accumulator buffer access with a K>0 check in GEMM epilogue code — when K=0 the MMA never wrote to the accumulator, so reading from it produces garbage; zero-fill the register fragment instead (zero is the identity element for accumulation)
- Register zero-fill is both correct and faster than unconditional global memory loads when the source buffer is known to be empty — fill(reg, 0) avoids issuing load instructions and bypasses the memory pipeline entirely
