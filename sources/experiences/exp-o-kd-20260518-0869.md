## Challenge

Optimization focus: Replaced scalar loop with warp-level vectorized copy for HND (strided heads) layout.
Problem before: The original reshape_and_cache_flash kernel handled both NHD and HND KV-cache layouts identically with a single scalar loop, using per-element division/modulo to compute head_idx 


## Solution
