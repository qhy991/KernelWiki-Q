fp32_flash_attention_qwen2_5_7b_f16_cache512

## Solution

One thread block per (batch, head) pair. Threads collaborate to compute attention scores across seq_len positions, then reduce for softmax normalization.

## Key Lessons

- Single-query attention has different data layout than standard Flash Attention
- Cross-warp reduction requires careful handling - __shfl_sync only works within a single warp
- Standard shared memory reduction is safer and simpler for block-level operations
- Memory layout [seq_len, num_heads, head_dim] requires specific indexing pattern
