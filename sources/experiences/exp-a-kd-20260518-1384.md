int8_mma_k32_repeated_single_vs_fourway

## Solution

rows 0-7 with B-half 0, (2) a[1]×b[0]→c[2,3] for rows 8-15 with B-half 0, (3) a[2]×b[1]→c[0,1] for rows 0-7 with B-half 1, (4) a[3]×b[1]→c[2,3] for rows 8-15 with B-half 1. Each m8n8k16 still takes a full tensor core MMA pipeline slot, so the four-way decomposition quadruples the instruction count. 

## Key Lessons

- The before source runs 100 loop iterations of one m16n8k32 MMA instruction (100 total MMA operations), while the after source runs 100 iterations of four m8n8k16 MMA instructions (400 total MMA operations).
