## Challenge

Optimization focus: Replace per-call cudaMalloc/cudaFree with caller-managed pre-allocated tensor buffers for tokens_cnts and cumsum in moe_align_block_size.
Problem before: The moe_align_block_size host function allocated tokens_cnts and cumsum buffers via cudaMalloc inside the function body and...


## Solution

The optimization eliminates four per-call synchronous cudaMalloc/cudaFree calls that force device-wide stream synchronization stalls each invocation.

- **speedup**: 90.9x
