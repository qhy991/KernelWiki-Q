KernelBench level_1 task_1: Square matrix multiplication C = A * B with N=2048

## Key Lessons

- Shared memory tiling with TILE_SIZE=16 is effective for matrix multiplication
- Float4 vectorization requires 16-byte alignment - easy to cause misaligned address errors
- Loop unrolling with #pragma unroll improves inner computation performance
- Basic CUDA kernel can achieve 2.90x speedup over PyTorch for this task
