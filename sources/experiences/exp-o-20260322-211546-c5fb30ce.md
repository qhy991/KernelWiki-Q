fp32_rms_norm_llama2_7b_hs128

## Solution

Shared memory weight caching - first warp loads weight to shared memory, all warps read from shared memory

## Key Lessons

- Shared memory for read-only weights provides no benefit when __ldg already caches in L1
- For small weight vectors (128 floats), L1 cache is sufficient
- Shared memory more useful for data reused across thread blocks or for read-write data
