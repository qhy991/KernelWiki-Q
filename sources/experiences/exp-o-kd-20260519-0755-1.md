## Solution

Add a bool kIsMLA template parameter to the kernel and wrap all V-cache pointer lookups, load_vec, and store_vec calls in if constexpr (!kIsMLA), so the compiler eliminates the entire V-cache code path at compile time when MLA mode is active, halving global memory traffic and removing per-layer pointer indirection overhead

## Key Lessons

- Use if constexpr with a template parameter to conditionally eliminate entire code paths at compile time — this avoids any runtime branch overhead and removes dead instructions from the generated binary, especially effective when a model variant (e.g., MLA) makes certain data transfers unnecessary
- When a kernel handles multiple data streams (K-cache + V-cache) and some workloads only need a subset, wrapping the optional stream in if constexpr halves memory traffic and pointer indirection overhead for those workloads with zero cost to the general path
