## Challenge

Implement kernel following the CUTLASS_device_function interface pattern with constraints: Target: SM 80 (Ampere); Shared memory limit: 48KB; Use uint16_t for 16-bit LDSM instruction support

## Key Lessons

- Interface pattern 'CUTLASS_device_function': the implementation follows the specified skeleton contract. Key constraints: Target: SM 80 (Ampere); Shared memory limit: 48KB
- Simple CUDA kernel implementation demonstrates the basic pattern: compute thread index, check bounds, perform work, write result. This pattern applies to any element-wise parallel operation
