## Challenge

Implement flash attention with mixed precision (FP32 query, F16 KV cache) and correct function signature (7 arguments: 3 tensors + 4 scalars)


## Solution

Naive attention kernel with two-pass softmax (max finding + accumulation)


### Key Changes

- Added 4 scalar parameters to forward() signature
- Used __half2float for F16 to FP32 conversion
- Two-pass softmax for numerical stability


## Key Lessons

- Test framework passes 7 arguments: query, key_cache, value_cache, batch_size, seq_len, num_heads, head_dim
- F16 KV cache requires __half2float conversion before FP32 computation
- Two-pass softmax (max finding + accumulation) ensures numerical stability
