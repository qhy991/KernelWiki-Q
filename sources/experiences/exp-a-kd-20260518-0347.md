## Key Lessons

- Although K_STAGE=4 only uses ~32 KB, the CUDA driver may reserve the full 96 KB shared memory budget per block, reducing the number of concurrent blocks per SM and thus occupancy.
- This illustrates that dynamic shared memory should only request the minimum needed, or the cudaFuncAttributeMaxDynamicSharedMemorySize should be set to the actual requirement rather than a fixed 98304.
