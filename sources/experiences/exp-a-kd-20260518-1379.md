## Challenge

On Turing's constrained 64KB shared memory, the before code allocates sh_a_size = 4 * (16 * 128) * 2 = 16384 bytes for INT8 activation — double what is actually needed


## Solution

The after code combines is_a_8bit=true (1 byte per element) with stages=2, computing sh_a_size = 2 * (16 * 128) * 1 = 4096 bytes.
