## Key Lessons

- Shared memory tiling with TILE_SIZE=32 works well for matrix multiplication
- PyTorch C++ Extension requires torch::empty() for output allocation, never cudaMalloc
- PYBIND11_MODULE must expose 'forward' function to match test framework expectations
- Symmetry of input matrices not exploited - could be future optimization opportunity
