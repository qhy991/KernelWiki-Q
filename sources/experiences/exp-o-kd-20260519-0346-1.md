## Solution

Fuse all five kernels into a single silu_bw_fused_kernel that loads x1, x2, dx4 once per element, computes sigmoid and x3 in registers, then reuses them to produce dx1, dx2, and x4 without any intermediate global memory writes, eliminating 5 kernel launches and 2 temporary buffers

## Key Lessons

- Fuse elementwise kernels that share intermediate values (sigmoid, x3) into a single kernel to eliminate redundant global memory writes/reads and kernel launch overhead — applies whenever multiple kernels operate on the same data elements and produce intermediates consumed by subsequent kernels
- Recompute inexpensive values (sigmoid) in registers rather than writing them to global memory for another kernel to read — recomputation is cheaper than global memory round-trip when the value fits in a register and the arithmetic cost is low
