## Challenge

Scalar half intrinsics (__hadd, __hmul) process one half element per instruction, missing the opportunity to use half2 packed arithmetic (hfma2) that doubles compute throughput by operating on two half values simultaneously


## Solution

Use recast<half2>() to reinterpret 8-element register tensors as 4-element half2 tensors, enabling the compiler to emit hfma2 instructions that perform two fused multiply-adds per instruction and halve the arithmetic loop iterations

## Key Lessons

- Packed half2 arithmetic (hfma2) doubles compute throughput for elementwise FP16 operations by processing two values per instruction — applies to any elementwise map where operations are lane-independent
- recast<half2>() is a zero-cost reinterpretation of register tensor layout that enables the compiler to use packed intrinsics without data movement or conversion overhead
