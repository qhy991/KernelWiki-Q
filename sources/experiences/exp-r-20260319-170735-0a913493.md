## Key Lessons

- For irregular GEMM shapes, always add boundary checks in shared memory tile loading
- Zero-pad out-of-bounds elements instead of skipping to maintain thread synchronization
- PyTorch C++ Extension requires torch::empty() for output allocation, never cudaMalloc
