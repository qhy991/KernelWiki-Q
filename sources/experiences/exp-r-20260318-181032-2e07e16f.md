## Key Lessons

- For C = A * B.T, access pattern is C[i][j] += A[i][k] * B[j][k]
- Shared memory tiling significantly improves GEMM performance by reducing global memory accesses
- TILE_SIZE=16 provides good balance between register usage and shared memory capacity
- TORCH_CHECK is required for error handling after kernel launch in PyTorch extensions
- PYBIND11_MODULE must use the same function name as expected by the test framework
