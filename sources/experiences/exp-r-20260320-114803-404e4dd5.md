## Key Lessons

- For C = A * B.T, access B as B[j][k] not B[k][j]
- Shared memory tiling with 16x16 tiles works well for M=1024, K=4096, N=2048
- PyTorch C++ Extension format requires torch::Tensor parameters and torch::empty() for allocation
