Metal matrix multiplication implementation on macOS

## Key Lessons

- Metal kernel functions cannot call other kernel functions
- Use constexpr instead of templates for compile-time constants in Metal kernels
- Threadgroup memory tiling with 16x16 tiles works correctly for GEMM optimization
- Both baseline and tiled implementations pass correctness verification
