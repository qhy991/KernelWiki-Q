## Solution

Fuse SiLU+mul and per-block quantization into a single kernel that computes the activation result in registers, performs shared-memory absmax reduction on the register value, broadcasts the scale via shared memory, and quantizes — eliminating the 64 MB intermediate global-memory round-trip and one kernel launch

## Key Lessons

- Fusing an activation kernel with a subsequent reduction/quantization kernel into a single kernel eliminates the intermediate global-memory round-trip — the activation result stays in registers and feeds directly into shared-memory reduction, which is critical for memory-bound operations where bandwidth dominates
- When the second kernel's input is exactly the first kernel's output and both operate on the same per-element or per-group granularity, fusion is straightforward: compute the activation in per-thread registers, then proceed directly to the shared-memory reduction within the same thread block
- Eliminating intermediate materialization removes not only global memory bandwidth but also one full kernel launch overhead and an extra cudaMalloc for the intermediate buffer, compounding savings for large tensors
