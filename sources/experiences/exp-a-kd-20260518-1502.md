## Solution

nel (exact GELU via ::erf) use CUDA intrinsics — tanhf and ::erf respectively — that map to hardware-assisted SFU operations with similar throughput. The tanh path computes x^3 (two extra FMULs) and a fused multiply-add before calling tanhf, while the erf path multiplies by ALPHA (one FMUL)
