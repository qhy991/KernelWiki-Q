## Solution

Fuse the absmax computation and quantization into a single kernel: compute per-token absmax via blockReduceMax, broadcast the result through shared memory with __syncthreads(), then quantize using multiplication by the precomputed reciprocal tmp_scale = 127.0f / block_absmax_val — eliminating the redundant second pass over the input tensor and one full kernel launch

## Key Lessons

- Fuse producer-consumer kernel pairs into a single kernel when the consumer's input (here, the scale) can be broadcast via shared memory — eliminates redundant global memory reads of the full input tensor and one kernel launch
- Use __shared__ memory + __syncthreads() to broadcast a block-level scalar (e.g., absmax) from the thread that computes it to all threads in the block, avoiding a round-trip through global memory
- Replace division by a per-block constant with multiplication by its reciprocal (tmp_scale = 127.0f / absmax) — multiplication has higher throughput than division on GPU FP units and the reciprocal is computed once per block
