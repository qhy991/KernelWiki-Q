## Solution

Capture the dropout mask into a compact 1-bit register array (cutlass::Array<cutlass::uint1b_t, ...>) immediately after generating Zij in shared memory, then apply dropout during DOIVJ using the register-resident mask instead of re-reading from shared memory, eliminating the second smem traversal and bounds-check branches

## Key Lessons

- Read shared-memory data into compact register-resident bitmasks immediately after generation when the same mask is consumed again later — avoids redundant smem traversals with 2D index computation and eliminates bounds-check branches from the hot path
- Use cutlass::uint1b_t (1-bit packed) for boolean masks in registers to minimize register pressure compared to storing full scalar_t values — the bitmask is N bits vs N×sizeof(scalar_t) bytes
