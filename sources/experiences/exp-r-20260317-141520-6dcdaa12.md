Implement RMS Norm operator using Metal on macOS with kernel.metal and metal_test.mm

## Key Lessons

- Metal kernel threadgroup design: use threadgroup_position_in_grid for grid-level indexing and thread_position_in_threadgroup for within-threadgroup indexing
- Threadgroup reduction must properly handle cases where not all threads are active (hidden_size < threads_per_threadgroup)
- Broadcast scalar values across threadgroup using shared memory with proper barrier synchronization
- For element-wise operations like RMS norm, each threadgroup can process one row/token with threads collaborating for reduction then writing individual elements
