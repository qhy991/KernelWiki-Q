Round 5/10 of AutoResearch: Implement Metal FP16 matrix multiplication with half4 vectorized loads optimization

## Key Lessons

- half4 vectorization loads 8 bytes per instruction instead of 2 bytes, improving memory bandwidth
- TILE_K must be set to 8 to ensure proper alignment for half4 loads (4 elements * 2 bytes = 8 bytes)
- Matrix A and Matrix B require different vectorized load patterns: consecutive elements vs consecutive rows
- Maintaining float accumulation preserves precision while using half4 for memory operations
