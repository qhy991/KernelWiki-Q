## Challenge

Implement kernel following the CUTLASS_device_function interface pattern with constraints: Target: SM 70+ (Volta) or newer with tensor core support; Uses 8x8x4 tensor operation instruction shape; Double precision (FP64) data type

## Key Lessons

- Interface pattern 'CUTLASS_device_function': the implementation follows the specified skeleton contract. Key constraints: Target: SM 70+ (Volta) or newer with tensor core support; Uses 8x8x4 tensor operation instruction shape
- CUTLASS-based implementations use template parameters for compile-time specialization: data types, layouts, tile sizes, and warp configurations are all template arguments rather than runtime parameters
