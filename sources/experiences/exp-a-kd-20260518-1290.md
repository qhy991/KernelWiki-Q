## Key Lessons

- The merged MinMax reduction eliminates one full BlockReduce pass and one __syncthreads() barrier, which is particularly valuable because the AZP kernel's before version had two serial reductions with a barrier between them.
- The key difference from the h16384 regression is the higher grid size (512 blocks vs 128), which provides more grid-level parallelism to compensate for the reduced per-block occupancy from 256 threads.
