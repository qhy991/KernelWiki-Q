## Solution

Wrap 4 floats into a __align__(16) array_t<float,4> struct and operate on packed elements so NVCC emits LDG.128/STG.128 instructions, transferring 16 bytes per transaction and reducing total memory transactions by 4x

## Key Lessons

- Reinterpreting a float array as a __align__(16) packed struct of 4 floats causes NVCC to emit 128-bit LDG/STG instructions, reducing memory transaction count by 4x — applies to any memory-bandwidth-bound kernel with contiguous, type-aligned access patterns
- Actual speedup is less than the theoretical 4x because loop iteration count and total active warps drop proportionally, while instruction overhead (loop counter, address computation) does not scale down — expect 2-3x for simple copy/fill kernels
