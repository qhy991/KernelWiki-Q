## Key Lessons

- Guard modulo-based branch conditions with a divisor-greater-than-1 check to prevent degenerate cases where x % 1 == 0 is always true — applies whenever modulo arithmetic is used for wave alignment or occupancy-based path selection
- In GPU kernel dispatch logic, an incorrect block count calculation propagates directly to grid size and SM occupancy, causing measurable performance degradation even when the computational kernel itself is unchanged
