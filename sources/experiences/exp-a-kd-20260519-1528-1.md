## Solution

Remove the outLogits output entirely: eliminate the float* outLogits parameter, the smemLogits shared memory array (saving 8KB of shared memory), and all logits reads/writes in both shortcut and full paths — halving global memory traffic in the memory-bound shortcut path

## Key Lessons

- Eliminating unnecessary output buffers (here outLogits) when downstream consumers don't need them directly halves global memory traffic in memory-bound kernels — applies whenever output data can be recomputed or is unused downstream
- Removing a parallel output stream also frees shared memory (8KB smemLogits removed here), which can improve occupancy or enable larger tile sizes in other shared-memory-heavy kernels
