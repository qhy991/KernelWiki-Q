aligned_128B

## Solution

ssue one ld.global.nc.v2.b64 + st.global.cg.v2.b64 pair). On the A800 (Ampere), the hardware memory subsystem already coalesces narrow per-lane accesses into wide memory transactions, so the explicit v2.b64 instruction does not reduce the actual memory bus traffic.

## Key Lessons

- The before code uses single 64-bit chunks with 16 active lanes (total_chunks=128/8=16, lanes 0-15 each issue one ld.global.nc.b64 + st.global.cg.b64 pair).
