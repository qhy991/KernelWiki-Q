## Challenge

Comparing the before and after source files, the before code declares __shared__ half q_smem[4][32*128] and k_smem[4][128], totaling 4*(4096+128)*2=33792 bytes of shared memory per block. It then iterates a compute loop 4 times with __hmul and __syncthreads per iteration, producing 4 sync barriers


## Solution

The after code reduces to __shared__ half q_smem[1][32*128] and k_smem[1][128] = 8448 bytes, with a single compute iteration and one sync barrier. - The critical difference is that the before kernel calls cudaFuncSetAttribute to opt-in to 163840 bytes of shared memory, which on A800 reduces L1 cache capacity and limits occupancy to fewer concurrent blocks per SM.
