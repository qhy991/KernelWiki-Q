## Key Lessons

- The after code replaces the preprocessor-guarded fp8_e4m3::scaled_vec_conversion with fp8::scaled_convert<K_vec, Quant_vec, KV_DTYPE>, which internally dispatches to scaled_vec_conversion with __NV_E4M3 via constexpr.
