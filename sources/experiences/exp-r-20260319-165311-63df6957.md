KernelBench level_1 task_7: Matrix multiplication C = A * B with small K dimension (M=N=16384, K=32)

## Key Lessons

- For small K dimension, tile entire K in shared memory to minimize global memory accesses
- BLOCK_SIZE=16 works well for 16384x16384 output matrices
- PyTorch C++ Extension format requires #include <torch/extension.h>, torch::Tensor parameters, torch::empty() allocation, and PYBIND11_MODULE
