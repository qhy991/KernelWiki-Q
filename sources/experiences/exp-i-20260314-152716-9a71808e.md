## Solution

## Key Lessons

- Test framework uses deterministic sampling (argmax) for reproducibility, not stochastic sampling
- Simple insertion sort works for small k but is inefficient for larger vocab sizes
- Performance bottleneck is O(vocab_size * k) per batch with insertion sort
