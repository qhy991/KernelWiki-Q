## Solution

Eliminate source tensor load and beta multiply entirely when beta=0 (OnlyAlphaScaling), replacing the three-step mul_source → mul_add_accum → relu pipeline with a two-step mul_accum → relu pipeline that reads only the accumulator array

## Key Lessons

- Eliminate dead data paths at compile time when scaling factors are zero — skipping the source tensor load halves global memory reads and removes unnecessary arithmetic in the epilogue pipeline
- Memory bandwidth savings from eliminating redundant loads are only measurable when the working set exceeds L2 cache capacity; at 64K fp32 elements (256KB) the data resides entirely in L2 and kernel launch overhead dominates, masking the bandwidth benefit
