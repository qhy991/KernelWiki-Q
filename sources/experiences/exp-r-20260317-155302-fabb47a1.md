## Key Lessons

- Metal reduce requires two-stage reduction: threadgroup reduction + CPU final reduction
- simd_shuffle can significantly improve performance for warp-level reduction
- Always verify expected values with simple test cases (all ones) before complex patterns
- Metal uses threadgroup memory (not shared memory like CUDA)
- threadgroup_barrier is required for synchronization in Metal
